- CommandLineTools

  - [`tee`](https://www.lesstif.com/lpt/linux-tee-89556049.html): 표준 입력(stdin)에서 읽어서 표준 출력(stdout)과 동시에 파일에 쓰는 명령줄 도구이다.

    - ##### example
    
        ```shell
        echo "hello" | tee example.txt
        ```
