- OS::Basic

  - ::Thread

    - `Green Thread`와 `OS Thread`의 차이.

    - OS Thread는 말그대로 OS단에서 생성 및 관리하는 Native Thread를 이야기한다. 반면, Green Thread는 1개의 Native Thread 내에서 동작하는, 프로그래밍적 Task Runtime이 존재해 코드 내에서 순차적으로 이루어지는 가상의 Thread를 이야기한다.
